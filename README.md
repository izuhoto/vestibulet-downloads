# Vestibulet Downloads

This repository hosts the distribution site content for **Vestibulet**, a drag-and-drop shelf app for macOS.

Vestibulet helps you move files and text between Finder and other apps more smoothly. While dragging, you can temporarily place items on a shelf, bring them back when needed, and keep frequently used resources organized in a unified library.

## Platform

- macOS 14 (Sonoma) or later

## Key Features

- Temporary shelf for files and text during drag-and-drop
- Quick shelf access from the screen edge, near the cursor, or via keyboard shortcuts
- Finder integration through Services / Quick Actions and shortcut-based intake
- History-based restore, path copying, Quick Look preview, and Finder reveal
- Unified library for frequently used files, folders, web pages, and app-related items
- Related-item navigation to move between connected work contexts
