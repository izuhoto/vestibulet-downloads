# Vestibulet v1.0.2 2026-07-01

**一時シェルフの ESC キー動作を修正**

ドラッグ中に表示される一時シェルフ（画面端・待機・修飾キー）で、ESC キーを押してもシェルフが閉じず、Finder 側にキー入力が届いてしまう不具合を修正しました。

- 一時シェルフ表示中に ESC を押すと、シェルフだけが閉じ、ドラッグは継続します
- 画面端・待機・修飾キーいずれの一時シェルフでも同様に動作します
- ESC で閉じたあと、同じドラッグ中は一時シェルフが再表示されません（マウスを離すとリセットされます）

※ この機能には「アクセシビリティ」のアクセス許可が必要です（システム設定 → プライバシーとセキュリティ → アクセシビリティ）。

# Vestibulet v1.0.1 2026-06-30

**ウィンドウが現在のワークスペース（Space）で開くようになりました**

- シェルフ・関連項目・ロケーションの各ウィンドウを再表示する際、前回閉じたワークスペースに画面が切り替わらず、**現在いるワークスペースに表示**されるようになりました。
- ウィンドウの**位置・サイズは従来どおり保持**され、ワークスペースだけが現在アクティブな場所に追従します。

# Vestibulet v1.0.0 2026-06-30

Vestibulet の初回リリースです。
ドラッグ中のファイルやテキストを一時的に置けるシェルフ、Finder 連携、履歴管理、そしてよく使う項目を整理する統合ライブラリの基本機能を利用できます。

## 主な機能
- ファイルやテキストを一時的に置けるドラッグ＆ドロップシェルフ
- 画面端、カーソル付近、キーボードショートカットからのシェルフ表示
- パスコピー、Quick Look、Finder 表示などのファイル操作
- ファイル、フォルダ、Web ページ、アプリ関連項目を管理できる統合ライブラリ
- 関連項目をたどれるナビゲーション
